# Add node-cron to package.json dependencies

Make sure to install the node-cron package:

```bash
npm install node-cron
```

Or add it to your package.json:

```json
{
  "dependencies": {
    "node-cron": "^3.0.3",
    "nodemailer": "^6.9.8",
    "jsonwebtoken": "^9.0.2",
    "axios": "^1.6.0",
    "@supabase/supabase-js": "^2.38.0",
    "cors": "^2.8.5",
    "dotenv": "^16.3.1",
    "express": "^4.18.2"
  }
}
```
